Indicador rsi:
zona de gatillo / ejecucion:
 - las confirmaciones de rsi (previa divergencia) ocurren en la
   zona de los 40-60 "zona clave" para confirmar el patron detectado.
 

Extremos del rsi:
 -casi siempre los extremos del indicador pueden estar marcado la formcaion de un nuevo patron 
  que se confirmar con posteriormente con una divegencia o figura. 



Invalidacion de señal:
 -este indicador no funciona para las tendencias.
 -en algunos casos cuando esta en tendencia, se puede usar para confirma zona de rebote, pero 
  se necesitan otros setups mas fuertes que lo validen.
